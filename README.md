## Introduction
`@bfun/cli` is the Standard Tooling for `bfunjs` Development.


## Documentation
visit [https://www.bfunjs.com/docs/cn/cli/index.html](https://www.bfunjs.com/docs/cn/cli/index.html)

```bash
# install
npm i @bfun/cli -g

# init project
mkdir app & cd app
bfun init [type]
```
